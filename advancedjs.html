Feature 1: Date Picker

In this example I will show you how a simple date input field can be turned into a more advance datepicker allowing the user to select dates from a dropdown calender.


Feature 2: 3D Model

In this example I will show you a 3D model of a phone than could be uised to show customers what courtesy phone they will be recieving, although it wont add any functionality to the web page, it will enhance the user experience for the customer.

Feature 3: Store Map & GeoLocation

A map could be added to the web page showing the user where store locations are, and by using thier goelocation it would potentially show them optimal routes and an estimate of how long it would take to get to each store.

Feature 4: Drag and Drop

This is another feature for the courtesy phone section that would let users drag what phone they want into a 'basket' and would replace the current method of using a drop down button click menu.

Feature 5: Web Storage

Finally this example shows how local storage can be used to store data and pass them to another webpage, or in this exapmple back to the user.
